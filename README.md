🚀 Just Launched My Udemy Clone Website! 🎓💻
"Learning by building is the best kind of learning!"

I'm thrilled to introduce my latest project — a fully functional static Udemy clone website created using pure HTML5 and CSS3.
This project allowed me to deeply explore layout design, responsive behavior, and front-end styling techniques.
Inspired by the original Udemy homepage, this clone replicates the look and feel of a professional course platform.

🔧 Tech Stack & Tools Used:
🧱 HTML5 – Semantic structure of the webpage

🎨 CSS3 – Styling, layout, colors, animations

🌐 Google Fonts – Elegant typography (Gabarito)

⭐ Font Awesome – For icons and UI enhancement

💡 VS Code – Development environment

🔁 Git & GitHub – Version control & collaboration

✨ Key Features:
🏠 Homepage layout modeled after Udemy for a real-world feel

📚 Recommended & Popular Courses section with price, title, instructor, and rating

🔍 Interactive Search Bar with Font Awesome search icon

🧭 Modern Navbar and category navigation pills

📱 Fully responsive layout (mobile, tablet, and desktop friendly)

🎴 Course Cards with hover animations for better UX

🖤 Dark-themed footer with multiple links and branding

🎯 What I Learned:
Structuring large web pages with clean HTML

Creating engaging user interfaces with only CSS

Organizing reusable CSS classes for maintainability

Implementing responsive design principles

UI/UX design inspirations from real-world websites
